## 🟦Griddle
__Griddle__ is a website you can draw simple pixelated art on, with basic features like picking colors.
This project is a W.I.P, built purely in Javascript.

This site was built using [Github Pages](https://plotchat.github.io/griddle_website/)
<br></br>

## 🛠 Installation

Install this project if you want to! There's nothing much so you can just simply clone it.
```bash
    git clone https://github.com/PlotChat/griddle_website.git
    cd griddle-website
```
